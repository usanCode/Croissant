# Croissant

This little webpage is dedicated to those who like
pastries in general and Croissant in particular.

Here is a [LIVE DEMO ](https://usanaselovescroissants.netlify.app/)

## 📌 Project Description

This project is a simple multi-page website dedicated to the croissant, a famous French pastry. The website introduces visitors to the history of the croissant, displays images of it, and allows users to contact the website owner through a contact form.

The goal of this project is to practice building a multi-page website and organizing content across different pages.

## 🌍 Pages Included
### 🏠 Home Page

The Home page presents the origins and history of the croissant.
Visitors can learn where the croissant comes from and how it became a popular pastry around the world.

### 📷 Photo Page

The Photo page displays several images of croissants.
It allows users to visually explore different types and presentations of this famous pastry.

### 📬 Contact Page

The Contact page includes a form that allows visitors to send a message.
Users can fill in their details and contact the website owner for questions or feedback.

## 🚀 How to Run the Project

1. Download or clone the repository.

2. Open the project folder.

3. Open the index.html file in your web browser.

4. The website will start from the Home page.
